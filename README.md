# RockPaperElements
RockPaperElements is a project by Martin Onga and Miron Golubev on HTML, CSS, and JavaScript that develops the classic ‘Rock, Paper, Scissors’ game.
This project uses technologies such as Node.js, Socket.io, and various secondary npm libraries

# Dependencies 

Node.js is required to run the project

npm install

npm i cors

npm i socket.io
